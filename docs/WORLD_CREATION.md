# Open-Pax — Creating Your World
==================================

## Обзор

Создание своего мира — ключевая фича Open-Pax. Ты можешь создать любую альтернативную реальность: от древних цивилизаций до далёкого будущего.

---

## Структура мира

### Минимальный мир

```json
{
  "id": "my-world",
  "name": "Мой мир",
  "description": "Описание мира",
  "start_date": "1951-01-01",
  "base_prompt": "Основная предпосылка мира",
  "historical_accuracy": 0.8,
  "regions": {},
  "blocs": {}
}
```

### Параметры

| Параметр | Обязательно | Описание |
|----------|-------------|----------|
| id | Да | Уникальный ID (латиница, без пробелов) |
| name | Да | Название мира |
| description | Нет | Краткое описание |
| start_date | Да | Дата начала игры (формат YYYY-MM-DD) |
| base_prompt | Да | Основная предпосылка мира |
| historical_accuracy | Нет | 0-1, насколько историчны события до start_date |

---

## Создание карты

### Способ 1: SVG Editor (Рекомендуемый)

1. Открой любой SVG редактор (Inkscape, Figma, Illustrator)
2. Нарисуй контуры регионов
3. Экспортируй каждый регион как path
4. Создай JSON с координатами

### Способ 2: Программный

```python
# Конвертировать SVG path в JSON
import json

regions = []

# Пример: простой квадратный регион
region = {
    "id": "moscow",
    "name": "Московское княжество",
    "path": "M 100 100 L 200 100 L 200 200 L 100 200 Z",
    "color": "#FF0000"
}
regions.append(region)

map_data = {
    "id": "my-map",
    "name": "My Map",
    "regions": regions,
    "width": 800,
    "height": 600
}

with open('map.json', 'w') as f:
    json.dump(map_data, f, indent=2)
```

## Продвинутые функции

### Связывание регионов (borders)

```json
{
  "regions": {
    "russia": {
      "id": "russia",
      "name": "Россия",
      "borders": ["germany", "china", "japan"]  // ID соседей
    }
  }
}
```

### Добавление объектов

```json
{
  "regions": {
    "moscow": {
      "id": "moscow",
      "name": "Москва",
      "objects": [
        {"type": "army", "name": "1-я армия", "x": 50, "y": 40},
        {"type": "university", "name": "МГУ", "x": 55, "y": 45}
      ]
    }
  }
}
```

### Типы объектов

| Тип | Описание | Визуал |
|-----|----------|--------|
| army | Армия/отряд | 🔴 Круг |
| factory | Завод | 🟠 Квадрат |
| university | Университет | 🟢 Круг |
| city | Город | ⬜ Ромб |
| port | Порт | 🔵 Треугольник |

## Импорт/Экспорт

### Экспорт мира

```bash
# Сохранить мир в файл
cp worlds/my-world.json worlds/my-world-backup.json
```

### Импорт мира

1. Положи файл в `public/maps/`
2. Загрузи через API или выбери в UI

---

## Лучшие практики

1. **Начни с простого** — 3-5 регионов достаточно для MVP
2. **Добавь историю** — base_prompt определяет атмосферу
3. **Тестируй** — запусти несколько ходов прежде чем делиться
4. **Итерации** — мир можно улучшать после первых игр
